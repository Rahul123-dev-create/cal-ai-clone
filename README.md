# Cal AI Clone — AI Calorie Tracker

A highly polished, premium mobile web-app clone of **Cal AI**, designed with a modern dark health aesthetic. This application allows users to simulate snapping a photo or typing in any meal to instantly receive dynamic calorie counts, calculated macronutrient breakdowns, and real-time dashboard tracking updates.

## Core Features Included

* **Dynamic AI Food Scan Engine:** Extracts keywords from user input descriptions or uploaded file names to programmatically calculate matching nutritional metrics.
* **Interactive Dashboard:** Features a dynamic circular progress ring tracking remaining daily calories alongside macro progress bars (Protein, Carbs, Fats) that update automatically upon meal confirmation.
* **Premium Processing States:** Utilizes animated laser scanning loops and progressive text status updates to maximize user feedback quality.
* **Analytics Trends & Progress Tracker:** Includes a weekly chart tracking nutritional history relative to targets and displays user streak counts.
* **User Customization:** Built-in settings form allowing configuration of weight, height, and custom daily calorie objectives.

## Tech Stack & Architecture

* **Frontend Framework:** Standalone HTML5, JavaScript (ES6+ State Management)
* **Styling System:** Tailwind CSS via CDN (Mobile Viewport Optimized Viewport)
* **Development Environment:** Constructed entirely using structural prompt architectures inside **Cursor AI Assistant**.

## AI Development Logs

As strictly required by the 8x Engineer competition parameters, the complete text history logs tracking the prompts, revisions, and structural decisions executed with the AI assistant during this build are securely preserved in the **`/ai-logs/`** folder of this repository.

## Local Setup Instructions

Since this is a fully compiled standalone frontend build, it requires zero server configuration or database setup to audit:
1. Clone or download this repository to your local machine.
2. Double-click the **`index.html`** file to launch the application immediately in any modern web browser.
